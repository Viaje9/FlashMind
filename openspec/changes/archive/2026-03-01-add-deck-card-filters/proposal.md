## Why

目前牌組內卡片清單只能以關鍵字搜尋，無法快速聚焦在「近期到期」或「尚未練習」等高優先卡片。新增篩選器可降低手動判讀成本，讓使用者更快找到當下要處理的卡片。

## What Changes

- 在牌組卡片清單的搜尋區塊右側新增 `Filter` 控制項。
- 提供三個可選篩選條件：
  - 七天內到期卡片
  - 尚未練習的新卡片
  - 三天內到期卡片
- 套用篩選後，卡片列表僅顯示符合條件的資料；未選擇篩選時維持現有列表行為。
- 篩選條件需可與既有搜尋功能共同作用（先搜尋再篩選，或等價結果）。

## Capabilities

### New Capabilities

- （無）

### Modified Capabilities

- `card-management`: 擴充牌組內卡片列表的查詢行為，新增可由使用者操作的篩選條件與對應列表顯示規則。

## Impact

- 規格：`openspec/specs/card-management/spec.md`（新增/調整需求情境）
- 前端：`apps/web` 牌組詳情頁卡片列表的查詢 UI 與列表過濾邏輯
- 後端/API：若現有卡片列表 API 尚未支援篩選參數，需擴充查詢參數與相對應資料過濾
