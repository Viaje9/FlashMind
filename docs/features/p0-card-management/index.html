卡片管理

概述

卡片 CRUD 功能，讓用戶新增、檢視、編輯、刪除快閃卡。

相關 User Stories

  • US-020：檢視牌組內卡片
  • US-022：新增卡片
  • US-024：編輯卡片
  • US-025：刪除卡片

驗收標準

US-020：檢視牌組內卡片

  • 顯示牌組摘要：新卡、待複習、建立時間、上次複習
  • 顯示卡片列表，每張卡片顯示正面文字與解釋摘要
  • 提供「開始學習」按鈕
  • 空狀態顯示引導新增卡片

US-022：新增卡片

  • 點擊右下角 + 按鈕開啟新增表單
  • 正面：輸入單字/片語/問題
  • 背面：支援多筆詞義區塊
  • 每筆詞義包含：中文解釋、英文例句、中文例句翻譯
  • 可新增/刪除詞義區塊
  • 英文例句提供播放語音按鈕
  • 頁首提供「取消」「儲存」
  • 儲存成功後返回卡片列表

US-024：編輯卡片

  • 點擊卡片列表項目的編輯按鈕進入編輯
  • 可修改正面與所有背面欄位
  • 儲存成功後返回卡片列表

US-025：刪除卡片

  • 點擊卡片列表項目的刪除按鈕
  • 刪除前顯示確認對話框
  • 刪除後更新牌組統計
