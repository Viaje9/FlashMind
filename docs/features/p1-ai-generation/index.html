AI 生成內容

概述

一鍵生成詞義與例句，大幅降低建卡成本。這是 FlashMind 的重要差異化功能。

相關 User Stories

  • US-023：AI 生成卡片內容

技術依賴

  • OpenAI API 或 Claude API

驗收標準

US-023：AI 生成卡片內容

  • 提供「AI 生成」按鈕
  • 輸入正面文字後可觸發生成
  • 生成過程顯示載入狀態
  • 生成結果自動填入背面欄位
  • 用戶可編輯/刪除生成內容
  • 生成失敗顯示錯誤訊息與重試選項

風險

風險緩解措施
AI 生成品質不佳用戶可編輯生成內容；持續優化 prompt
AI API 成本過高設定每日生成上限；考慮本地模型
