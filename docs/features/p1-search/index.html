搜尋功能

概述

提供牌組與卡片的搜尋功能，改善用戶體驗。少量資料時可省略。

相關 User Stories

  • US-011：搜尋牌組
  • US-021：搜尋卡片

驗收標準

US-011：搜尋牌組

  • 提供搜尋輸入框
  • 即時過濾牌組列表（輸入時即過濾）
  • 無結果時顯示「找不到符合的牌組」

US-021：搜尋卡片

  • 提供搜尋輸入框
  • 搜尋範圍包含正面文字與背面解釋
  • 即時過濾卡片列表
