# FlashMind PRD（依據 prototype）

## 產品背景
FlashMind 是一款以快閃卡為核心的學習工具，聚焦於「建立 → 學習 → 複習」的循環。此 PRD 以 `prototype/` 現有畫面為範圍，整理功能與需求，作為下一階段開發的規格依據。

## 目標
- 讓使用者能快速建立牌組與卡片，並進入學習模式。
- 支援每日學習計畫（新卡/複習數）與學習提醒。
- 提供清楚的學習進度與牌組管理能力。

## 目標使用者
- 有語言/考試/專業知識背誦需求的自學者。
- 需要每日小量學習與穩定複習的使用者。

## 產品範圍（對應 prototype 畫面）
- 歡迎/登入頁面
- 主介面（牌組列表）
- 牌組新增（學習計畫設定）
- 牌組內卡片列表
- 新增/編輯快閃卡（含 AI 生成、例句）
- 學習模式（正面/背面）
- 設定介面

## 核心流程
1. 進入歡迎頁 → 註冊/登入/第三方登入。
2. 進入主介面 → 檢視牌組、搜尋、建立新牌組。
3. 進入牌組 → 檢視牌組概況、搜尋卡片、管理卡片。
4. 新增/編輯卡片 → 建立正面與多筆詞義/例句。
5. 進入學習模式 → 顯示卡片正面 → 翻面 → 回答「知道/不知道」。
6. 透過設定頁調整學習偏好與一般選項。

## 功能需求

### 1. 歡迎/登入
- 提供「開始使用」「我已有帳號」入口。
- 提供第三方登入（Apple/Google/Facebook）。
- 顯示服務條款與隱私權政策連結。

### 2. 主介面（牌組列表）
- 搜尋牌組。
- 顯示牌組卡片狀態：新卡、待複習、完成。
- 顯示學習進度條與百分比。
- 牌組右側提供「開始學習」入口。
- 右下角 + 按鈕建立新牌組。

### 3. 新增牌組（學習計畫設定）
- 輸入牌組名稱。
- 設定每日新卡數（預設 20，步進 5）。
- 設定每日複習數（預設 100，步進 10）。
- 顯示提示：可於牌組設定調整與複習量提醒。
- 提供「儲存設定」「取消」。

### 4. 牌組內卡片列表
- 顯示牌組摘要（新卡、待複習、建立時間、上次複習）。
- 「開始學習」按鈕。
- 牌組內卡片搜尋。
- 卡片列表項目：
  - 顯示正面文字與中文解釋摘要。
  - 提供編輯與刪除操作。
- 右下角 + 按鈕新增卡片。

### 5. 新增/編輯快閃卡
- 正面：輸入單字/片語/問題（字數統計）。
- 背面：支援多筆詞義/例句區塊。
- AI 生成：一鍵生成詞義與例句。
- 每筆詞義包含：中文解釋、英文例句、中文例句翻譯。
- 英文例句提供播放語音。
- 允許新增/刪除詞義區塊。
- 頁首提供「取消」「儲存」。

### 6. 學習模式（正面/背面）
- 顯示牌組名稱與子標題（例：日常對話）。
- 顯示進度（12/50）與進度條。
- 卡片正面：顯示單字/問題。
- 卡片背面：顯示中文解釋、例句與翻譯。
- 卡片操作：收藏、播放發音。
- 回答選項：知道 / 不知道（作為學習紀錄輸入）。

### 7. 設定介面
- 管理帳戶。
- 登出帳戶。

## 非功能需求（NFR）
- 行動裝置優先，支援 360px 以上寬度。
- 深色模式支援。
- 主要互動保持 60fps（滑動翻卡、按鈕）。
- 無障礙：文字對比與可點擊區域 44px 以上。

## 指標
- 新增牌組轉換率（從主介面進入新增並完成儲存）。
- 每日學習啟動率（當日開始學習的使用者比例）。
- 卡片新增完成率（進入新增卡片 → 儲存）。
- 7 日留存率。

## 假設與限制
- AI 生成詞義/例句功能需有可用模型與配額。
- 語音播放為外部 TTS 服務。
- 多語系目前以繁體中文為主。

## 開放問題
- 學習模式中「知道/不知道」的演算法採用 FSRS。
- 牌組「子標題」（例：日常對話）來源為標籤還是描述？
- AI 生成是否可批次套用多張卡？
- 匯出學習數據格式（CSV/JSON）與範圍？

## 里程碑建議
- M1：帳號登入 + 牌組 CRUD + 卡片 CRUD。
- M2：學習模式 + 紀錄 + 進度。
- M3：設定頁 + 提醒 + 匯出。
