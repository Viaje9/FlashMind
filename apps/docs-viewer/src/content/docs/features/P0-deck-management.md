# 牌組管理

## 概述

牌組 CRUD 功能，讓用戶建立、檢視、編輯、刪除牌組。

## 相關 User Stories

- US-010：檢視牌組列表
- US-012：建立新牌組
- US-013：編輯牌組設定
- US-014：刪除牌組

## 驗收標準

### US-010：檢視牌組列表

- [x] 顯示所有牌組的列表
- [x] 每個牌組顯示：名稱、新卡數、待複習數、完成數
- [x] 顯示學習進度條與百分比
- [x] 提供「開始學習」快捷入口
- [x] 空狀態顯示引導建立牌組

### US-012：建立新牌組

- [x] 點擊右下角 + 按鈕開啟建立表單
- [x] 必填：牌組名稱
- [x] 可設定每日新卡數（預設 20，步進 5，範圍 5-100）
- [x] 可設定每日複習數（預設 100，步進 10，範圍 10-500）
- [x] 顯示提示：「可於牌組設定調整」
- [x] 儲存成功後導向牌組卡片列表
- [x] 取消時返回主介面，不儲存任何資料

### US-013：編輯牌組設定

- [x] 可修改牌組名稱
- [x] 可修改每日新卡數與複習數
- [x] 儲存後立即生效於下次學習

### US-014：刪除牌組

- [x] 刪除前顯示確認對話框
- [x] 確認對話框說明將刪除所有卡片與學習紀錄
- [x] 刪除後返回主介面
