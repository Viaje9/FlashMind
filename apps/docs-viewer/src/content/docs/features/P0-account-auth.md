# 帳號與登入

## 概述

提供用戶註冊、登入、登出功能，是使用 FlashMind 的前提。

## 相關 User Stories

- US-001：新用戶註冊
- US-002：現有用戶登入
- US-003：登出帳號

## 驗收標準

### US-001：新用戶註冊

- [x] 支援 Email 註冊
- [x] 支援第三方登入（Google）
- [x] 註冊成功後自動登入並導向主介面
- [x] 顯示服務條款與隱私權政策連結

### US-002：現有用戶登入

- [x] 支援 Email + 密碼登入
- [x] 支援第三方登入（Google）
- [x] 登入成功後導向主介面
- [x] 登入失敗顯示錯誤訊息

### US-003：登出帳號

- [x] 在設定頁提供登出選項
- [x] 登出後清除本地登入狀態
- [x] 登出後導向歡迎頁
