# 帳號與登入

## 概述

提供用戶註冊、登入、登出功能，是使用 FlashMind 的前提。

## 相關 User Stories

- US-001：新用戶註冊
- US-002：現有用戶登入
- US-003：登出帳號

## 驗收標準

### US-001：新用戶註冊

- [ ] 支援 Email 註冊
- [ ] 支援第三方登入（Google）
- [ ] 註冊成功後自動登入並導向主介面
- [ ] 顯示服務條款與隱私權政策連結

### US-002：現有用戶登入

- [ ] 支援 Email + 密碼登入
- [ ] 支援第三方登入（Google）
- [ ] 登入成功後導向主介面
- [ ] 登入失敗顯示錯誤訊息

### US-003：登出帳號

- [ ] 在設定頁提供登出選項
- [ ] 登出後清除本地登入狀態
- [ ] 登出後導向歡迎頁
