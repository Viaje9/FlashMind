# 學習模式

## 概述

核心學習功能，包含翻卡學習、滑動評分、學習進度追蹤。

## 相關 User Stories

- US-030：開始學習
- US-031：翻卡學習
- US-032：回答評分
- US-034：完成學習

## 驗收標準

### US-030：開始學習

- [ ] 顯示牌組名稱
- [ ] 顯示學習進度（例：12/50）與進度條
- [ ] 依 FSRS 演算法排序卡片（新卡 + 待複習）
- [ ] 優先顯示待複習卡片，再顯示新卡
- [ ] 每日學習數量不超過設定上限

### US-031：翻卡學習

- [ ] 初始顯示卡片正面（單字/問題）
- [ ] 點擊/滑動可翻到背面
- [ ] 背面顯示：中文解釋、例句、例句翻譯
- [ ] 翻卡動畫流暢（60fps）

### US-032：回答評分

- [ ] 翻到背面後支援三種滑動手勢：
  - 右滑 = 知道
  - 上滑 = 不熟
  - 左滑 = 不知道
- [ ] 滑動後記錄結果並進入下一張卡片
- [ ] 根據回答更新 FSRS 排程（知道的間隔最長，不知道的間隔最短）
- [ ] 「不知道」的卡片會在本次學習中再次出現
- [ ] 滑動時顯示視覺回饋（卡片移動、背景色變化）
- [ ] 支援返回按鈕，讓用戶修改評分

### US-034：完成學習

- [ ] 所有卡片完成後顯示結束畫面
- [ ] 顯示本次學習統計：學習張數、各評分分布（知道/不熟/不知道）
- [ ] 提供「返回牌組」按鈕
